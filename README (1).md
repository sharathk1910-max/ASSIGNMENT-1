 Library Management System 

  Project Description
This **Library Management System** is a simple Python-based mini project designed for beginners.  
It allows users to manage book records using a menu-driven program.

The project is suitable for:
- College mini projects
- Python practice
- Lab assignments
- Viva and exams



 Features
- Add new book records
- View all available books
- Search books by category
- Save book records to a file
- Preloaded with **20 book records**
- Simple and user-friendly menu system



 Technologies Used
- Python 3
- File Handling
- Lists & Dictionaries
- Functions
- Exception Handling
